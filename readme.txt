A WordPress plugin that provides a wrapper for the OpenAI API with a ChatGPT-like interface.

== Description ==

OpenAI Wrapper allows you to integrate OpenAI's chat functionality directly into your WordPress site. It provides a ChatGPT-like interface through a simple shortcode [openai-wrapper].

Features:
* Easy setup through WordPress admin
* ChatGPT-like interface with dark theme
* Support for multiple OpenAI models
* Session-based conversation persistence
* Secure API communication

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/openai-wrapper` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Settings->OpenAI Wrapper screen to configure the plugin
4. Place the shortcode [openai-wrapper] in any post or page where you want the chat interface to appear

== Frequently Asked Questions ==

= How do I get an OpenAI API key? =

Visit OpenAI's website and create an account to obtain your API key.

= How do I use the chat interface? =

Simply add the shortcode [openai-wrapper] to any post or page where you want the chat interface to appear.
